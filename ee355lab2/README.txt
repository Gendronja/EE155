Program Summary:
	Matrix multiplication using two square matrices and calculates runtime of code. to run, ensure that there is an input.txt file with the contents of two square matrices. Any size square matrices will work. Outputs two text files: output.txt, which contains the square C resulting matrix, and output_time.txt, which contains the runtime in seconds.
